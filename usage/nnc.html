nnc¶

nnc will extract Non-Neighbour-Connections from your Eclipse grid as pairs of ijk indices together with their associated transmissibilities.

See also the usage/trans module, which can extract all transmissibilities, not only non-neigbour connections.

from ecl2df import nnc, EclFiles

eclfiles = EclFiles('MYDATADECK.DATA')
dframe = nnc.df(eclfiles)

Alternatively, the same data can be produced as a CSV file using the command line

ecl2csv nnc MYDATADECK.DATA --verbose --output nnc.csv

It is possible to add xyz coordinates for each connection (as the average of the xyz for each of the cells involved in a connection pair) as extra columns.

If you only want vertical connections, add the option --pillars or -vertical, or set pillars=True if using the Python API (ecl2df.nnc.df())
